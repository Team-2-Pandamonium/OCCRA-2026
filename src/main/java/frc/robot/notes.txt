CONTROL SCHEME:
Driver:
left stick - left side wheels
right stick - right side wheels
RB - turbo mode
LB - slow mode

Opperator:
Y - Level 3
B - Level 2
A - Level 1
X - HP station
Level_Button+RB - level rotate
LB - elevator 0
DPad_Up - elevator up ("fast")
DPad_Up_Right - elevator up ("slow")
DPad_Down - elevator down ("fast")
DPad_Down_Right - elevator down ("slow")
RT - intake
LT - outtake
Left Stick - manipulator long arm
Right Stick - manipulator short arm
